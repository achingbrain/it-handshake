Intro

Installable via npm install --save it-handshake, it can also be used directly in the browser.

Quick Example

'use strict'
/* eslint no-console: ["error", { allow: ["log"] }] */

const pipe = require('it-pipe')
const duplex = require('it-pair/duplex')
const handshake = require('./')

;(async () => {
  // Create connected duplex streams
  const [client, server] = duplex()
  const clientShake = handshake(client)
  const serverShake = handshake(server)

  clientShake.write('hello')
  console.log('client: %s', await serverShake.read())
  // > client: hello
  serverShake.write('hi')
  serverShake.rest() // the server has finished the handshake
  console.log('server: %s', await clientShake.read())
  // > server: hi
  clientShake.rest() // the client has finished the handshake

  // Make the server echo responses
  pipe(
    serverShake.stream,
    (source) => (async function * () {
      for await (const message of source) {
        yield message
      }
    })(),
    serverShake.stream
  )

  // Send and receive an echo through the handshake stream
  pipe(
    ['echo'],
    clientShake.stream,
    (source) => (async () => {
      for await (const bufferList of source) {
        console.log('Echo response: %s', bufferList.slice())
        // > Echo response: echo
      }
    })()
  )
})()

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save it-handshake

You can then require() it-handshake as normal:

const itHandshake = require('it-handshake')

In the Browser

It-handshake should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of it-handshake, including index.js and index.min.js, are included in the /dist folder. It-handshake can also be found on unpkg.com under

  • https://unpkg.com/it-handshake/dist/index.min.js
  • https://unpkg.com/it-handshake/dist/index.js
